# Apex スタイルガイド

## はじめに

## プラクティス
### アクセス
- アクセス修飾子を適切に使用する
### クラスメソッド・クラス変数
- [SHOULD] インスタンスからクラスメソッド、クラス変数にアクセスしない。
### 定数
- 数値定数は-1、0、1を除いて直接コーディングしない。
### 変数代入
- **[MUST]** 1つのステートメントで複数の変数に同じ値を代入しない。
```
fooBar.fChar = barFoo.lchar = 'c'; // AVOID!
```
### メソッド定義
- オーバーライドは避ける。どうしても必要な場合はコメントに目的・意図を書く。

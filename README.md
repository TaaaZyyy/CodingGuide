## Keywords
- __"MUST"__ 必須事項。開発者はmasterにマージするときに遵守していること。
- __"SHOULD"__ 推奨事項。クリティカルではないが、従うことが推奨される。
- __"ok"__ （サンプルコード）推奨される書き方。
- __"avoid"__ （サンプルコード）避けるべき書き方。
- __"ng"__ （サンプルコード）禁止される書き方。
